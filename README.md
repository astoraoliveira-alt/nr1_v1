# NR-1 Compliance Agent Landing Page

Landing page de alta conversão para o Agente de IA Conversacional focado na adequação à NR-1.

## Tecnologias

- **React 18**
- **Vite**
- **Tailwind CSS**
- **Lucide React** (Ícones)
- **Framer Motion** (pronto para uso, embora animações CSS tenham sido priorizadas para performance)

## Como rodar

1. Instale as dependências:
   ```bash
   npm install
   ```

2. Rode o servidor de desenvolvimento:
   ```bash
   npm run dev
   ```

3. Acesse `http://localhost:5173`

## Estrutura do Projeto

- `src/components`: Componentes da página (Hero, Form, Sections)
- `src/index.css`: Estilos globais e configuração do Tailwind
- `tailwind.config.js`: Tema personalizado (Cores, Fontes, Animações)

## Funcionalidades

- Formulário de Lead com máscara de telefone e validação
- Dashboard Mockup responsivo
- Layout Mobile-First
- Design System "Glassmorphism" e Gradientes modernos
